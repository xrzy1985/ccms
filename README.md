### Customer Content Management System

## Angular 12, SASS, json-server for local db

# Pre-reqs

vscode

# Instructions

Open the project folder, the folder with the package.json file
Open a terminal in that folder
npm i -g
json-server db.json
ng serve -o --port=9000
